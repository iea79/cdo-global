CDO. Электронная информационная образовательная среда (CDO.ЭИОС) - портал для организации дистанционного обучения и взаимодействия обучающихся, преподавателей и сотрудников университета. Портал представляет собой интеграционное решение платформы Moodle и соответствующей конфигурации 1С, которые взаимодействуют между собой посредством API.

Решение позволяет построить единое цифровое пространство университета, объединив существующие информационные системы. Система CDO.ЭИОС разработана в соответствии с требованиями ФГОС 3++ и ФЗ-152 и регулярно модернизируется при появлении новых нормативно-правовых актов.

Масштабируемость
Система может быть заинтегрирована с любой информационной системой, имеющей открытое API
Гибкость
Мы не ограничиваем наших клиентов имеющимися сервисами. Система всегда может быть достроена и доработана под Ваши потребности
